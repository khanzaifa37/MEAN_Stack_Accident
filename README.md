# MEAN_Stack_Accident
To run locally:
1. Install node.
2. Install angular.
3. Start mongodb locally.
4. cd express-server.
5. Run npm install.
6. Run node server.js.
7. cd ../angular-app.
8. Run npm install.
9. Run ng serve.

To get docker image: https://hub.docker.com/repository/docker/singhaniket406/mean-stack-application
